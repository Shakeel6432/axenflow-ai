# AxenFlow AI — Next.js Website

Premium AI Automation Agency website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Pages

- `/` — Home (Hero, Services, Portfolio, FAQ, Client Form, etc.)
- `/services` — All services
- `/portfolio` — Case studies
- `/about` — About us
- `/contact` — Contact + requirement form
- `/privacy` — Privacy Policy
- `/terms` — Terms of Service

## Images

Images are linked via junction from the original Techin template:

public/images → ../techinhtml.netlify.app/assets/images

## Run locally

```bash
cd axenflow-ai
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```
